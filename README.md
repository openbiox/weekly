## 如何搜索

1、使用 [Sourcegraph.com](https://sourcegraph.com/github.com/openbiox/weekly) 进行搜索。（推荐方法）

2、使用 GitHub 自带的搜索，但只支持搜索英文单词。

3、将这个仓库克隆到本地，然后在仓库目录使用下面的命令。

```bash
$ grep -nri [搜索词] issues | cat --number
```

比如，搜索 bash 相关内容。

```bash
$ grep -nri bash issues | cat --number
```
